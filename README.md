# Gryps Frontend React Test

## Setup

#### YARN

```
// Install dependencies
yarn

// Run development server with hot reload
yarn start

// Build for production
yarn build

// Lint files
yarn lint
```

#### NPM

```
// Install dependencies
npm install

// Run development server with hot reload
npm run start

// Build for production
npm run build

// Lint files
npm run lint
```
